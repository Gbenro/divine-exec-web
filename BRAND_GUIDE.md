# Divine Executive Chauffeurs - Brand Style Guide

## Brand Essence
**Tagline:** Where punctuality meets luxury

**Brand Promise:** Premium executive transportation that combines reliability, professionalism, and comfort for Chicago's business and leisure travelers.

**Brand Personality:**
- Professional and dependable
- Elegant and sophisticated
- Approachable yet exclusive
- Detail-oriented

---

## Color Palette

### Primary Colors
- **Navy Dark** `#1a1a2e` - Primary background, text headers
- **Navy Medium** `#16213e` - Secondary backgrounds, gradients
- **Gold** `#c9a227` - Accent color, CTAs, highlights
- **Gold Light** `#d4b84a` - Hover states, secondary accents

### Neutral Colors
- **Text Primary** `#333333` - Body text
- **Text Light** `#666666` - Secondary text
- **White** `#ffffff` - Backgrounds, light text
- **Off-White** `#f8f9fa` - Alternate section backgrounds
- **Border** `#e0e0e0` - Dividers and borders

### Usage Guidelines
- Navy communicates trust, professionalism, and executive appeal
- Gold represents luxury, premium service, and excellence
- Use gold sparingly for maximum impact on CTAs and key elements
- Maintain high contrast for accessibility

---

## Typography

### Primary Font: Playfair Display
**Usage:** Headings, logo, taglines
**Weights:** 400 (Regular), 600 (Semi-Bold), 700 (Bold)
**Character:** Elegant, sophisticated serif that conveys luxury and tradition

### Secondary Font: Inter
**Usage:** Body copy, navigation, forms, buttons
**Weights:** 300 (Light), 400 (Regular), 500 (Medium), 600 (Semi-Bold)
**Character:** Modern, clean, highly readable sans-serif

### Type Scale
- **H1:** 2.5rem - 4rem (responsive clamp) - Hero headlines
- **H2:** 1.75rem - 2.5rem (responsive clamp) - Section titles
- **H3:** 1.25rem - Service/card headings
- **Body:** 16px base - Optimal readability
- **Small:** 0.85rem - 0.95rem - Captions, metadata

### Typography Best Practices
- Line height: 1.6 for body text, 1.2 for headings
- Letter spacing: 0.5px - 1px for uppercase text
- Use Playfair Display italic for taglines and quotes

## Visual Style

### Design Principles
1. **Clean & Elegant:** Generous whitespace, uncluttered layouts
2. **Premium Feel:** Subtle shadows, smooth transitions, refined details
3. **Professional:** Grid-based layouts, consistent spacing
4. **Accessible:** High contrast, readable fonts, clear hierarchy

### Components Style

#### Buttons
- Primary: Gold background, navy text, uppercase, rounded corners (4px)
- Hover: Lighter gold, slight lift effect (-2px transform)
- Padding: 14px 32px (standard), 10px 24px (nav)
- Letter spacing: 1px

#### Cards
- Background: White on off-white, subtle shadows
- Border radius: 8px
- Padding: 32-40px
- Hover: Lift effect (-8px transform), enhanced shadow

#### Sections
- Alternating backgrounds: White / Off-white / Navy / Gold
- Padding: 100px vertical (desktop), 80px (tablet), 60px (mobile)
- Max width container: 1200px

### Iconography
- Style: Simple, clean, emoji-based for now
- Color: Gold for icons on light backgrounds
- Size: 1.5rem - 3rem depending on context

## Voice & Tone

### Brand Voice
- **Professional:** Clear, competent, trustworthy
- **Refined:** Elegant language without being pretentious
- **Personal:** Warm and attentive, not cold or distant
- **Confident:** Direct and assured about service quality

### Writing Guidelines
- Use "we" and "our" to build relationship
- Focus on client benefits, not just features
- Active voice for clarity and impact
- Short, scannable paragraphs for web content

### Example Phrases
✅ "Elevate Your Journey"
✅ "Where punctuality meets luxury"
✅ "Always on time, always professional"
✅ "Premium transportation experiences"

❌ "Best car service ever!"
❌ "Cheap rides"
❌ "The #1 choice" (without proof)

## Logo & Branding Elements

### Current Logo
- Diamond symbol (♦) in gold
- "Divine Executive Chauffeurs" in Playfair Display
- Clean wordmark, no decorative elements

### Logo Usage
- Minimum size: 180px wide
- Clear space: Equal to height of diamond symbol
- Never distort or rotate
- Always maintain gold/navy color relationship

---

## UI Patterns

### Spacing System
- Base unit: 8px
- Scale: 8, 16, 24, 32, 40, 60, 80, 100px
- Consistent application maintains rhythm

### Shadows
- Small: `0 2px 4px rgba(0, 0, 0, 0.08)` - Subtle elevation
- Medium: `0 4px 12px rgba(0, 0, 0, 0.12)` - Cards, buttons
- Large: `0 8px 24px rgba(0, 0, 0, 0.16)` - Modals, hover states

### Transitions
- Standard: `all 0.3s ease`
- Use for: hover states, transforms, color changes
- Keep subtle and refined, never jarring

---

## Mobile Considerations

### Breakpoints
- Desktop: 1200px+ (default)
- Tablet: 768px - 992px
- Mobile: Below 768px
- Small mobile: Below 480px

### Mobile-First Principles
- Touch-friendly targets (minimum 44x44px)
- Simplified navigation (hamburger menu)
- Stacked layouts on small screens
- Maintain readability and hierarchy

---

## Accessibility

### Standards
- WCAG 2.1 Level AA compliance target
- Color contrast ratio: Minimum 4.5:1 for text
- Focus indicators on interactive elements
- Semantic HTML structure

### Testing
- Keyboard navigation support
- Screen reader compatible
- Responsive text sizing
- Alternative text for images
